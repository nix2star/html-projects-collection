# Intermediate Level HTML Projects

This folder contains intermediate-level HTML projects.  

All projects are coded in **HTML and CSS**, showing a higher level of structure and styling.  

Focus: combining sections, navigation, forms, media, and accessibility.

Skills demonstrated:
- Lists, links, forms  
- Semantic HTML tags (`<section>`, `<article>`, `<header>`, `<footer>`)  
- Improved CSS layout (side-by-side elements, padding, margins)

## Projects
1. Multi-Page Personal Portfolio – add navigation between “Home,” “About,” and “Projects.”


2. Company Landing Page – structure a homepage with a hero section and features.


3. Registration Form – use various input types, radio buttons, checkboxes, and <fieldset>.


4. Blog Homepage – list multiple articles with images and read-more links.


5. Event Invitation Page – include date, time, location, and RSVP form.


6. Travel Destination Page – include map (iframe), gallery, and key details.


7. Music Player Page – embed audio files with <audio> and captions.


8. Video Showcase Page – display videos with <video> and custom captions.


9. Wikipedia-Style Article Page – use <aside>, <nav>, and table of contents links.


10. Restaurant Menu Page – organize food items in categories using tables and lists.
