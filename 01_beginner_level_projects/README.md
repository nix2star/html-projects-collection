# Beginner Level HTML Projects

This folder contains beginner-level HTML projects.  

All projects are coded in **HTML and CSS** (basic styling).  

Focus: mastering structure, semantic tags, text, lists, links, images, and basic forms.

Projects in this folder demonstrate:
- Basic HTML structure (`<html>`, `<head>`, `<body>`)  
- Headings, paragraphs, images, etc
- Simple CSS styling (colors, fonts, spacing)

## Projects
1. Personal Introduction Page – simple page with heading, paragraph, and image.


2. Favorite Quotes Page – display your favorite quotes using <blockquote> and <cite>.


3. Simple Portfolio Page – show name, hobbies, and contact section using semantic tags.


4. Recipe Page – include ingredients (lists), steps (ordered list), and image of the dish.


5. HTML Resume – use headings, paragraphs, lists, and contact links.


6. Product Page – display a product with its image, description, and price.


7. Basic Photo Gallery – use multiple <img> tags inside <figure> and <figcaption>.


8. Feedback Form – create a basic form with input fields and a submit button.


9. Simple Blog Post Layout – use <article>, <header>, <section>, and <footer>.


10. HTML Email Template (no CSS) – use only tables for layout.
