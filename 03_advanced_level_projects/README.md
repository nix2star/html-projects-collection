This folder contains advanced-level HTML projects.  

All projects are coded in **HTML and CSS** and responsive design and layout is included.  

Focus: responsive design, complex forms, and accessibility improvements.

Skills demonstrated:
- Flexbox and CSS Grid layouts  
- Responsive design for desktop, tablet, and mobile  
- Interactive elements using CSS hover and transitions
